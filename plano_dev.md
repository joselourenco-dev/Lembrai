# 🎂 App de Aniversários — Plano de Desenvolvimento

> **Mentor:** Programador Sênior  
> **Aluno:** Você  
> **Linguagem:** Kotlin  
> **IDE:** Android Studio  
> **Duração estimada:** 6 etapas progressivas

---

## ✅ ETAPA 0 — Configuração do Ambiente
**Objetivo:** Ter tudo pronto para começar a codar.
- [ ] Baixar e instalar o Android Studio
- [ ] Criar o primeiro projeto Android
- [ ] Entender a estrutura de pastas do projeto
- [ ] Rodar o app "Hello World" no emulador

**O que você vai aprender:** O que é uma IDE, o que é um emulador, estrutura de um projeto Android.

---

## 📋 ETAPA 1 — Estrutura e Tela Principal
**Objetivo:** Criar a tela inicial que lista os aniversários.
- [ ] Entender o que é uma Activity e um Layout
- [ ] Criar uma lista (RecyclerView) para exibir os aniversários
- [ ] Entender XML (linguagem de interface visual)

**O que você vai aprender:** Activities, Layouts XML, RecyclerView, Adapter.

---

## ➕ ETAPA 2 — Tela de Adicionar Aniversário
**Objetivo:** Criar o formulário para cadastrar uma pessoa.
- [ ] Criar uma nova tela (Activity)
- [ ] Adicionar campos: Nome, Data, Categoria (Família/Amigo)
- [ ] Navegar entre telas (Intent)
- [ ] Capturar e validar dados do formulário

**O que você vai aprender:** Múltiplas Activities, Intent, EditText, Spinner, validação de dados.

---

## 💾 ETAPA 3 — Salvar os Dados (Banco de Dados Local)
**Objetivo:** Guardar os aniversários no celular permanentemente.
- [ ] Entender o que é um banco de dados
- [ ] Usar o Room Database (banco de dados do Android)
- [ ] Criar a entidade "Pessoa" (nome, data, categoria)
- [ ] Operações: salvar, listar, editar e deletar

**O que você vai aprender:** Room Database, DAO, Entity, Coroutines (processamento em segundo plano).

---

## 🔔 ETAPA 4 — Notificações e Destaques
**Objetivo:** Avisar o usuário quando um aniversário estiver chegando.
- [ ] Ordenar a lista por proximidade do aniversário
- [ ] Destacar aniversários do dia e da semana
- [ ] Criar notificações automáticas (alarme no dia do aniversário)

**O que você vai aprender:** Manipulação de datas, WorkManager, NotificationManager.

---

## ✏️ ETAPA 5 — Editar e Excluir
**Objetivo:** Permitir gerenciar os contatos salvos.
- [ ] Abrir tela de edição ao clicar em um nome
- [ ] Atualizar dados no banco
- [ ] Confirmar exclusão com diálogo

**O que você vai aprender:** Reutilização de telas, AlertDialog, atualização no banco de dados.

---

## 🎨 ETAPA 6 — Polimento Visual (UI/UX)
**Objetivo:** Deixar o app bonito e agradável de usar.
- [ ] Aplicar cores e ícones personalizados
- [ ] Adicionar ícone do app
- [ ] Melhorar tipografia e espaçamentos
- [ ] Testar no celular físico

**O que você vai aprender:** Material Design, temas, recursos de imagem, build de APK.

---

## 🏁 Final
Ao terminar as 6 etapas, você terá:
- Um app Android funcional e completo
- Conhecimento real de Kotlin
- Base sólida para criar outros apps
